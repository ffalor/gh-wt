## gh wt rm

Remove a worktree and its associated branch

### Synopsis

Remove a worktree and its associated branch. Will prompt if there are uncommitted changes (unless --force is used).

```
gh wt rm [worktree-name] [flags]
```

### Options

```
  -h, --help   help for rm
```
